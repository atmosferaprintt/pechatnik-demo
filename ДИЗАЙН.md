# Дизайн-направление CRM «ПЕЧАТНИК»

Зафиксировано 2026-07-13 по рефам от Рабии (дашборд Crextio, HR-система). Все новые экраны и `*-variants.html` делаем в этой стилистике.

## Общий вайб

Светлый, тёплый, «мягкий» интерфейс: кремово-жёлтые градиенты, большие скругления, много воздуха. Контраст — точечными тёмными (почти чёрными) карточками и пилюлями. Никакого корпоративного синего.

## Палитра (черновые токены — уточним при вёрстке вариантов)

| Токен | Значение | Где |
|---|---|---|
| `--bg` | `#f6f3ec` → градиент к `#fdf6dd` (тепло-жёлтый снизу/справа) | фон приложения |
| `--card` | `#ffffff` / `#faf8f2` | карточки |
| `--accent` | `#f7d64a` (тёплый жёлтый) | акценты, прогрессы, активные элементы |
| `--dark` | `#1d1d1f` | тёмные карточки-акценты, активная вкладка, кнопки |
| `--text` | `#1d1d1f` | основной текст |
| `--muted` | `#8a8a85` | вторичный текст |
| `--line` | `#e8e4d9` | разделители, рамки |
| Доп. серый | `#d9d9d6` | неактивные чипы/бары |

## Формы и компоненты

- **Скругления крупные:** карточки 20–28px, кнопки и чипы — полные «пилюли» (999px).
- **Навигация** — горизонтальная лента-пилюля сверху; активная вкладка = тёмная пилюля с белым текстом, остальные — текст на светлом.
- **Карточки** — белые, мягкая тень (`0 8px 24px rgba(0,0,0,.06)`), без жёстких рамок.
- **Тёмные акцентные блоки** — 1–2 на экран (например, виджет «сверка дня»), тёмный фон + белый текст + жёлтые отметки.
- **Крупные цифры** — ключевые показатели очень крупно (48–64px, light/medium), подпись мелкая под ними.
- **Прогресс-бары** — толстые (28–36px) скруглённые сегменты: жёлтый / тёмный / серый; допустима диагональная штриховка для «пустой» части.
- **Аватары/инициалы** — круглые.
- **Иконки** — ТОЛЬКО свой набор линейных SVG (`src/Icon.jsx`, stroke 1.9, наследуют цвет текста): `<I n="search" size={14} />`. **Эмодзи в интерфейсе НЕ используем** (решение Рабии 2026-07-16). Типографские глифы ✓ ✕ → ← ↩ ± допустимы. Новую иконку — добавлять в Icon.jsx в том же стиле.

## Типографика

Округлый геометрический гротеск. Берём **Manrope** (или Onest) с Google Fonts, локально в public/fonts при деплое. Заголовки — крупные, обычной жирности (не bold-капс), цифры — tabular.

## Настроение по разделам

- Задачи (канбан) — светлые колонки, карточки-пилюли, жёлтая подсветка дедлайнов
- Финансы — тёмная карточка сверки с разницей ±, жёлтые суммы
- Аналитика — крупные цифры + мягкие бары, как на рефе «Progress / Time tracker»

Рефы лежат в корне проекта (2 PNG от 2026-07-13, стиль Crextio).
